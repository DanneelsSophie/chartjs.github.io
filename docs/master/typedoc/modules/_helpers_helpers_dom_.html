readUsedSize

  • readUsedSize(element: any, property: any): number
    • Defined in helpers/helpers.dom.js:194

    The "used" size is the final value of a dimension property after all calculations have been performed. This method uses the computed style of element but returns undefined if the computed style is not expressed in pixels. That can happen in some cases where element has a size relative to its parent and this last one is not yet displayed, for example because of display: none on a parent node.
